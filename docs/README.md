## ğŸ“Š Quick Reference

### Phase Comparison

| Feature | Phase 1 | Phase 2 |
|---------|---------|---------|
| **Generation** | Random | Neural (Transformer) |
| **Success Rate** | 2-5% | 10-30% |
| **Learning** | None | Yes |
| **Curriculum** | No | Yes |
| **Script** | `train.py` | `train_neural.py` |
| **Documentation** | [Quick Start](getting-started/quickstart.md) | [Phase 2 Docs](phase2/) |

### File Locations

| What | Where |
|------|-------|
| **Data** | `AutoConjecture/data/` |
| **Checkpoints** | `data/checkpoints/` |
| **Logs** | `data/logs/` |
| **Scripts** | `AutoConjecture/scripts/` |
| **Source Code** | `AutoConjecture/src/` |

### Key Commands

```bash
# Phase 1: Random generation
python scripts/train.py

# Phase 2: Neural generation (CPU)
python scripts/train_neural.py

# Phase 2: Neural generation (GPU)
python scripts/train_neural.py --device cuda

# Resume from checkpoint
python scripts/train_neural.py --resume data/checkpoints/generator_epoch_5.pt

# Run tests
pytest tests/test_neural_components.py -v
```
